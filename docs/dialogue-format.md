# 双人对话输入格式

当前推荐的最简单格式：

```text
A|今天这期我们先聊最近市场最值得关注的三件事。
B|第一件事还是油价和地缘政治。
A|第二件事是美国市场重新定价高利率更久的风险。
B|第三件事是中美关系和全球供应链预期。
```

## 规则

- 每行一个发言轮次
- `A|` 表示角色 A
- `B|` 表示角色 B
- 竖线后面是该角色要念的文本

## 当前验证成功的配置

- 角色 A：`zh_female_mizai_uranus_bigtts`
- 角色 B：`zh_male_dayi_uranus_bigtts`
- volume：A=100，B=85

## 说明

这是当前已经验证“听感明显更舒服”的双人配置。
